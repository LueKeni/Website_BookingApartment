TECHNICAL RULES FOR CHAT:
1. Real-time: Use Socket.io. Do not use API polling for messages.
2. UI:
   - Chatbox on ApartmentDetail should be a toggleable popup (sticky bottom-right).
   - Use a scrollable message list with "Auto-scroll to bottom" when new messages arrive.
   - Differentiate Bubble colors: User (Blue), Agent (Gray).
3. Security: Only participants of a conversation can fetch its messages (check `authMiddleware`).
4. Optimization: Load only the last 20-50 messages initially, use pagination for older messages.
