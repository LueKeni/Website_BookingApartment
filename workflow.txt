USER ROLES: USER, AGENT, ADMIN.

WORKFLOWS:
1. Authentication:
   - All roles use the same login/register form but are redirected to different dashboards based on their role.

2. Normal User Workflow:
   - Browse apartments (filter by rent/sale, price, area).
   - View apartment details and agent info.
   - Click "Book Appointment" -> Select Date/Time -> Submit.
   - Add apartments to "Favorites".
   - View booking history.
   - Leave a Review (1-5 stars) for the Agent ONLY AFTER a booking status is COMPLETED.

3. Agent (Cộng tác viên) Workflow:
   - Dashboard: View statistics (total listings, successful deals, response rate).
   - Manage Apartments: CRUD operations for their own listings.
   - Manage Schedule: Set available days/times.
   - Manage Bookings: View incoming requests, update status (PENDING -> CONFIRMED / CANCELLED -> COMPLETED).

4. Admin Workflow:
   - Dashboard: Overview of system metrics.
   - Manage Users: View all users and agents. Ban/Unban accounts.
   - Moderate Listings: Hide or delete inappropriate apartment listings.
