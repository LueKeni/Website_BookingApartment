CHAT WORKFLOW:

1. Starting a Chat:
   - User views `ApartmentDetail` -> Clicks "Chat with Agent".
   - System checks if a `Conversation` already exists for this User + Agent + Apartment.
   - If no: Create new `Conversation`. If yes: Open existing one.
   - Redirect or Open a Mini Chatbox at the bottom right.

2. Real-time Messaging:
   - Use Socket.io to join a "Room" based on `conversationId`.
   - When User/Agent sends a message, emit 'send_message' event.
   - Server saves message to DB and broadcasts to the room.

3. Message Center (Inbox):
   - User/Agent can see a list of all conversations.
   - Display `lastMessage` and a "New Message" badge if `isRead` is false.
   - Clicking a conversation opens the full `ChatWindow`.
